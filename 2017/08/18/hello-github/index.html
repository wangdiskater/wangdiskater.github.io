hello github!

By 王邸 at: 2017-08-18

前戏

好久不玩github了突然不会了。老司机一不小心就翻车，还是要好好学习啊兄弟。不学习就翻车啦。以前用github上传文件的时候都是使用图形化界面。然后就好多细节的操作都不知道。今天用命令行终于链接上了。还是要记录一下。不过最重要的是：知道每一步的作用。先一顿操作在各种博客弄好之后，去到git官网走一波…..

回归

今天是2017-08-23终于把我这个破电脑修好了。自己手动换了电脑的主板。这样也就是对电脑有了更新的认识了。以后哪里坏的就修哪里咯。笔记本电脑主要还是集成的，很多引线的链接都是通过开关式的按压来完成的。并不像台式机是拔插的。所以需要格外的小心。昨天差点就把键盘给gg了。辛亏没有压好，总之电脑弄好了之后做啥都带劲 哈哈。
