开头

为了去培训，最近在写JAVA基础的50题。这些题目不算太难，有一些思路总结一下，涉及到一些JAVA的知识。使用输入输出等等。那我们就开始吧！

50题精选

古典问题：有一对兔子，从出生后第3个月起每个月都生一对兔子，小兔子长到第三个月后每个月又生一对兔子，假如兔子都不死，问每个月的兔子对数为多少？
答： 这个问题看起来很复杂 但是把每个月兔子数写出来就很容易发现规律 1 1 2 3 5 8 13 21….从第三个月开始每个月都是前两个月兔子的加和。这里就用递归来实现就好；

自己写起来就比较难受，题目参考答案直接交换两个数这就变得很简单了

题目：打印出杨辉三角形（要求打印出10行如下图）
思路：通过分析得出杨辉三角的一个规律：
第n行就有n个数，第一个和第n个数都是为1的，如果我们用i表示行数；j表示列数
arr[i][j]=arr[i-1][j]+arr[i-1][j-1]

题目：输入两个正整数m和n，求其最大公约数和最小公倍数。
思路：求最大公因数有三种每种分别实现以下：相减法穷举法和辗除法
最小公倍数：两个数相乘除以最大公因数即为最小公倍数
总结： 相减法和辗除法原理并不清楚。而穷举法比较像之前中学数学求公因数的方法

题目：一个数如果恰好等于它的因子之和，这个数就称为”完数”。例如6=1＋2＋3.编程找出1000以内的所有完数。
思路：一开始思路错了以为是：先找出因数然后再+1，后来才知道原来是对完数的定义错了
完数的定义是所有的约数除去自身之外的所有的数的和。
例如：28 = 1X2X2X7 2,2,7是因数 。 但是完数要的是所有能被整除的数有 1,2,4,7,14
先求出所有能被整除的数放在数组，然后再加和
