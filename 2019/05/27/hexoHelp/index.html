介绍

这是一个简单的hexo使用技巧的总结。两年不玩了，最近从新搞出来，是不要要总结一下呢

一些指令的用法

生成一个新的日志文档:在blog的目录下面输入:
hexo new filename

要清除本地生成的文件在blog的目录下面输入:
hexo clean

要重新生成文件在blog的目录下面输入:
hexo g

想发布在git在blog的目录下面输入:
hexo d

当然可以通过help来查看还有什么命令可以用，因为太久不用了。所以可能多了很多功能都没用
