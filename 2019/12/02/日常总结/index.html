遇到问题

1)客户端向服务端请求拿不到数据问题
说明：在网关服务通过Micronaut的内置的HttpClient请求别的服务的时候，发现能请求到服务，发送回来的数据是存在的，但是接收到是一个空的数组。

解决：经过检查发现之前在服务的Controller对应方法上的@Get()注解出了问题，之前编写demo添加了”produces = MediaType.TEXT_PLAIN”，导致回传数据不是以json的格式，从而接收不到。Micronaut默认是json传输的，所以不写produces默认能返回json数据。

2)模糊查询问题
说明：使用Micronaut Data JDBC，显示的JDBC查询,即使用@Query()注解自定义sql语句时候，直接在语句里面写百分号报错。例如：
@Query(“select id,name,price from book b where b.title like %:title% limit 5”)
List findBooks(String title);
解决：在传入title的时候，把模糊查询使用的符号拼接在title上，再调用Repository的方法传入拼接好的title即可。

3）window查看端口占用问题

netstat -ano | findstr 8080
taskkill /pid 8080 /f
